# AI2025実験の分析結果

## 1. 意思決定時間の分析結果（time_analysis.R）

### 記述統計
- AI条件の平均意思決定時間: 1.77秒（SD = 1.01）
- コントロール条件の平均意思決定時間: 1.67秒（SD = 1.01）
- AI条件の中央値: 2.0秒
- コントロール条件の中央値: 1.5秒

### 統計的検定（混合効果モデル）
- 固定効果:
  - ベースライン（Control条件）: 1.63秒
  - AI条件の効果: +0.09秒（SE = 0.06, t = 1.43）
- ランダム効果:
  - 参加者間のばらつき（SD）: 0.09
  - 残差のばらつき（SD）: 0.82
- 結果の解釈: AI条件とControl条件の間で意思決定時間に統計的に有意な差は見られなかった（t = 1.43）

### データの品質
- 総サンプル数: 787
- 有効回答数:
  - AI条件: 430
  - Control条件: 357
- 外れ値処理:
  - 0秒以下のデータを除外
  - 上位1%を外れ値として除外

### 主な知見
1. AI条件の参加者は平均して約0.09秒長く意思決定に時間をかける傾向
2. この差は比較的小さく、個人差（SD = 0.09）と同程度の大きさ
3. 両条件とも意思決定時間の分布は右に歪んでいる（密度プロット参照）

## 2. 配分額の分析結果（payoff_avg_analysis.R）

### 制限ありサンプルでの分析

  - 平均配分額: 660.34円（SE = 17.76, n = 174）
- コントロール条件:

### 主な知見
1. 全サンプルにおいて、AI条件とコントロール条件の間の配分額の差は約9.31円（コントロール条件が高い）だが、統計的に有意ではない
2. 制限ありサンプルでは、その差は約15.21円（コントロール条件が高い）に拡大するが、やはり統計的に有意ではない
3. 両条件とも、制限ありサンプルでは全サンプルよりも平均配分額が低下する傾向が見られた
4. 標準誤差は制限ありサンプルで大きくなっており、より大きなばらつきが示唆される

## 4. 社会的選好パラメータの分析結果（social_preference_analysis.R）

### 全サンプルでの分析

#### パラメータ推定結果
- AI条件:
  - α (不利な不平等回避) = -0.016 ± 0.125
  - β (有利な不平等回避) = 0.050 ± 0.082
  - λ (選択の感度) = 7.657 ± 1.681

- Control条件:
  - α (不利な不平等回避) = -0.072 ± 0.085
  - β (有利な不平等回避) = 0.132 ± 0.055
  - λ (選択の感度) = 11.339 ± 1.946

#### 条件間の差（AI - Control）と統計的検定
- Δα = 0.056（不利な不平等回避の差）
  - z = 0.369, p = 0.712（統計的に有意な差なし）
- Δβ = -0.082（有利な不平等回避の差）
  - z = -0.834, p = 0.405（統計的に有意な差なし）
- Δλ = -3.682（選択感度の差）
  - z = -1.432, p = 0.152（統計的に有意な差なし）

### 制限ありサンプルでの分析（利己的選択が明確な状況）

#### パラメータ推定結果
- AI条件:
  - α (不利な不平等回避) = -0.500 ± 0.129
  - β (有利な不平等回避) = -0.340 ± 0.159
  - λ (選択の感度) = 3.887 ± 1.232

- Control条件:
  - α (不利な不平等回避) = -0.365 ± 0.137
  - β (有利な不平等回避) = 0.040 ± 0.076
  - λ (選択の感度) = 11.046 ± 2.197

#### 条件間の差（AI - Control）と統計的検定
- Δα = -0.135（不利な不平等回避の差）
  - z = -0.718, p = 0.473（統計的に有意な差なし）
- Δβ = -0.380（有利な不平等回避の差）
  - z = -2.158, p = 0.031（統計的に有意な差あり）
- Δλ = -7.159（選択感度の差）
  - z = -2.843, p = 0.004（統計的に有意な差あり）

### 主な知見

1. **全サンプルでの分析**
   - AI条件とControl条件の間で社会的選好パラメータに統計的有意差なし
   - 両条件とも、有利な不平等回避（β）が不利な不平等回避（α）より強い傾向

2. **制限ありサンプルでの分析**
   - AI条件で有意に低い有利な不平等回避（β）
     - AI条件: -0.340、Control条件: 0.040
     - 差は統計的に有意（p = 0.031）
   - AI条件で有意に低い選択の一貫性（λ）
     - AI条件: 3.887、Control条件: 11.046
     - 差は統計的に有意（p = 0.004）
   - 不利な不平等回避（α）は有意差なし

3. **分析手法の違いが示唆すること**
   - 利己的選択が明確な状況でAIの影響がより顕著に
   - 特に有利な立場での社会的選好と選択の一貫性に影響
   - より制御された状況で社会的選好の違いが検出可能

## 6. 選好と報酬額の関係性分析結果（payoff_preference_analysis.R）

### データの基本統計量
#### 報酬額の分布
- 最小値: 140.0円
- 第1四分位: 470.0円
- 中央値: 720.0円
- 平均値: 679.7円
- 第3四分位: 870.0円
- 最大値: 1060.0円

#### 条件ごとの報酬額統計
- Control条件:
  - 平均報酬額: 685円
  - 標準偏差: 228円
  - 範囲: 290円 - 1060円
- AI条件:
  - 平均報酬額: 675円
  - 標準偏差: 239円
  - 範囲: 140円 - 1060円

### 重回帰分析結果
#### モデル概要
- 従属変数: 報酬額（player.payoff）
- 独立変数: 各種選好、実験条件、性別、年齢

#### 係数推定値
1. 選好タイプの影響
   - 利己的選好: +18.11円 (SE = 10.91, p = 0.097)
   - 平等性選好: -0.44円 (SE = 10.90, p = 0.968)
   - 効率性選好: +2.27円 (SE = 9.09, p = 0.803)
   - 競争的選好: -1.07円 (SE = 8.91, p = 0.905)

2. 条件の効果
   - AI条件: -9.22円 (SE = 16.86, p = 0.584)

3. 統制変数
   - 性別: +0.52円 (SE = 17.46, p = 0.976)
   - 年齢: +0.13円 (SE = 5.93, p = 0.982)

#### モデルの適合度
- 重決定係数（R²）: 0.00594
- 調整済み重決定係数: -0.00290
- F統計量: 0.672 (df = 7, 787)
- p値: 0.696

### 多重共線性の診断
すべての変数のVIF（分散拡大係数）は2未満：
- 利己的選好: 1.59
- 平等性選好: 1.59
- 効率性選好: 1.11
- 競争的選好: 1.06
- 実験条件: 1.02
- 性別: 1.09
- 年齢: 1.06

### 主な知見
1. 選好と報酬額の関係について：
   - 利己的選好が報酬額に対して弱い正の影響（限界的有意性：p < 0.10）を示した
   - その他の選好タイプ（平等性、効率性、競争性）は報酬額に有意な影響を示さなかった

2. 実験条件の影響：
   - AI条件は報酬額に対して小さな負の効果（-9.22円）を示したが、統計的に有意ではなかった
   - この結果は、条件間の報酬額の差が実質的に小さいことを示唆している

3. モデルの説明力：
   - 全体として、モデルの説明力は非常に低い（R² < 0.01）
   - これは、報酬額の変動が本研究で考慮した要因以外の影響を受けている可能性を示唆している

4. 個人特性の影響：
   - 性別や年齢による報酬額への有意な影響は検出されなかった
   - これらの人口統計学的要因は、本実験における意思決定に実質的な影響を与えていない可能性が高い
