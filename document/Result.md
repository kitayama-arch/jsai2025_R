# AI2025実験の分析結果

## 1. 意思決定時間の分析結果（time_analysis.R）

### 記述統計
- AI条件の平均意思決定時間: 1.77秒（SD = 1.01）
- コントロール条件の平均意思決定時間: 1.67秒（SD = 1.01）
- AI条件の中央値: 2.0秒
- コントロール条件の中央値: 1.5秒

### 統計的検定（混合効果モデル）
- 固定効果:
  - ベースライン（Control条件）: 1.63秒
  - AI条件の効果: +0.09秒（SE = 0.06, t = 1.43）
- ランダム効果:
  - 参加者間のばらつき（SD）: 0.09
  - 残差のばらつき（SD）: 0.82
- 結果の解釈: AI条件とControl条件の間で意思決定時間に統計的に有意な差は見られなかった（t = 1.43）

### データの品質
- 総サンプル数: 787
- 有効回答数:
  - AI条件: 430
  - Control条件: 357
- 外れ値処理:
  - 0秒以下のデータを除外
  - 上位1%を外れ値として除外

### 主な知見
1. AI条件の参加者は平均して約0.09秒長く意思決定に時間をかける傾向
2. この差は比較的小さく、個人差（SD = 0.09）と同程度の大きさ
3. 両条件とも意思決定時間の分布は右に歪んでいる（密度プロット参照）

## 4. 社会的選好パラメータの分析結果（social_preference_analysis.R）

### パラメータ推定結果

#### AI条件:
- α (不利な不平等回避) = -0.016 ± 0.125
- β (有利な不平等回避) = 0.050 ± 0.082
- λ (選択の感度) = 7.657 ± 1.681

#### Control条件:
- α (不利な不平等回避) = -0.072 ± 0.085
- β (有利な不平等回避) = 0.132 ± 0.055
- λ (選択の感度) = 11.339 ± 1.946

### 条件間の差（AI - Control）と統計的検定
- Δα = 0.056（不利な不平等回避の差）
  - z = 0.369, p = 0.712（統計的に有意な差なし）
  - 効果量: d = 0.503（中程度の効果量）
- Δβ = -0.082（有利な不平等回避の差）
  - z = -0.834, p = 0.405（統計的に有意な差なし）
  - 効果量: d = -1.159（大きい効果量）
- Δλ = -3.682（選択感度の差）
  - z = -1.432, p = 0.152（統計的に有意な差なし）
  - 効果量: d = -2.076（非常に大きい効果量）

### パラメータの分布特性
1. 不利な不平等回避（α）:
   - AI条件: 95%CI [-0.311, 0.187], SD = 0.125
   - Control条件: 95%CI [-0.284, 0.053], SD = 0.085
   - AI条件でより広い分布を示す

2. 有利な不平等回避（β）:
   - AI条件: 95%CI [-0.150, 0.170], SD = 0.082
   - Control条件: 95%CI [0.008, 0.226], SD = 0.055
   - Control条件でより安定した正の値を示す

3. 選択の感度（λ）:
   - AI条件: 95%CI [5.028, 11.814], SD = 1.681
   - Control条件: 95%CI [8.233, 15.000], SD = 1.946
   - 両条件で広い分布を示すが、明確な水準の差がある

### パラメータ間の関係分析
1. AI条件の特徴:
   - αとβが広い範囲に分布（α: -0.3〜0.2, β: -0.15〜0.17）
   - パラメータ間の相関が比較的弱い
   - より多様な意思決定パターンを示唆
   - 個人差が大きく、予測が難しい選好パターン

2. Control条件の特徴:
   - より狭い範囲に集中（α: -0.28〜0.05, β: 0.01〜0.23）
   - βが主に正の値に集中
   - より一貫した社会的選好パターン
   - 予測可能性の高い意思決定傾向

3. 条件間の比較解釈:
   - Control条件では比較的一貫した社会的選好を示す
   - AI条件では意思決定パターンが多様化
   - この違いは、AI条件が参加者の意思決定プロセスに質的な影響を与えている可能性を示唆

### 主な知見
1. AI条件では、Control条件と比較して：
   - 不利な不平等への回避度が高いが、有意差なし（Δα > 0, p = 0.712）。中程度の効果量（d = 0.503）
   - 有利な不平等への回避度が低いが、有意差なし（Δβ < 0, p = 0.405）。大きい効果量（d = -1.159）
   - 選択の感度が低い（有意差なし）（Δλ < 0, p = 0.152）。非常に大きい効果量（d = -2.076）
2. 両条件とも、有利な不平等回避（β）が不利な不平等回避（α）より強い
3. Control条件の方が選択の一貫性が高い傾向にある（λが大きい）。効果量は非常に大きい
